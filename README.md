# TEMPLATE_GO

## 项目结构

```bash
template_go    
├── config            // 配置文件
├── example           // 使用实例
├── src               // 源代码
├── tests             // 测试代码
├── Dockerfile        // Dockerfile文件
├── go.mod            // go mod 包管理,自带依赖 go_utils
├── main.go           // 主启动文件
├── start.sh          // 自动打包编译运行 docker 脚本,启动的是主启动文件, 使用时候需要修改启动端口
```
